# دليل الاستخدام - نظام بناء التقارير الديناميكي

## نظرة عامة

نظام ويب مرن لتصميم تقارير PDF احترافية بدون تعقيد. يتيح لك اختيار قالب جاهز، تعديل المحتوى، إدخال البيانات، ومعاينة التقرير قبل تصديره كملف PDF.

## البدء السريع

### 1. تثبيت المتطلبات

```bash
npm install
```

### 2. تشغيل النظام

```bash
npm run dev
```

افتح المتصفح على [http://localhost:3000](http://localhost:3000)

## خطوات إنشاء تقرير

### الخطوة 1: اختيار القالب

- في الصفحة الرئيسية، اختر أحد القوالب الجاهزة:
  - **تقرير تنفيذي**: للمجلس التنفيذي
  - **تقرير الأداء**: لمؤشرات الأداء
  - **تقرير المشاريع**: لمتابعة المشاريع
  - **تقرير مالي**: للتقارير المالية

### الخطوة 2: إدخال البيانات

في لوحة **البيانات** (الجانب الأيمن):
- أدخل القيم للمتغيرات المستخدمة في القالب
- أو استخدم محرر JSON لإدخال البيانات دفعة واحدة

**مثال:**
```json
{
  "report_period": "الربع الأول 2024",
  "executive_summary": "ملخص تنفيذي هنا...",
  "key_results": "النتائج الرئيسية..."
}
```

### الخطوة 3: تعديل الأقسام

في لوحة **التحرير**:
1. اضغط على قسم في المعاينة لتحديده
2. استخدم الأدوات لتعديل:
   - المحتوى
   - المحاذاة (يمين، وسط، يسار)
   - حجم الخط
   - لون النص
   - الهوامش

### الخطوة 4: إضافة أقسام جديدة

من الشريط الجانبي الأيسر:
- اضغط على نوع القسم المراد إضافته
- سيتم إضافته إلى التقرير
- يمكنك تعديله من لوحة التحرير

### الخطوة 5: المعاينة والتصدير

- استخدم زر **معاينة** لإخفاء/إظهار عناصر التحرير
- اضغط **تصدير PDF** لتنزيل التقرير النهائي

## المتغيرات في القوالب

استخدم `{{variable_name}}` في محتوى القسم لربط البيانات:

```
{{report_period}}
{{executive_summary}}
{{key_results}}
```

ثم أدخل القيم في لوحة البيانات.

## أنواع الأقسام

- **عنوان**: لعناوين الأقسام الرئيسية
- **نص**: للنصوص العادية
- **رأس**: للعناوين الرئيسية
- **مسافة**: لإضافة مسافة بين الأقسام

## نصائح الاستخدام

1. **ابدأ بقالب جاهز**: يوفر الوقت والجهد
2. **استخدم المتغيرات**: لتسهيل تحديث البيانات
3. **اختبر المعاينة**: قبل التصدير النهائي
4. **احفظ البيانات**: يمكنك نسخ JSON للاستخدام لاحقاً

## بنية المشروع

```
reports/
├── app/              # صفحات Next.js
├── components/       # مكونات React
├── lib/             # المكتبات والخدمات
├── templates/       # القوالب الجاهزة
├── types/           # تعريفات TypeScript

```

## التخصيص

### إضافة قالب جديد

1. أنشئ ملف في `templates/`
2. استخدم نفس البنية الموجودة
3. أضفه إلى `templates/index.ts`

### تخصيص الألوان

عدّل ملف `tailwind.config.js` لتغيير الألوان الرئيسية.

## الدعم

للمساعدة أو الاستفسارات، راجع الكود المصدري أو اقرأ التعليقات في الملفات.
